## 功能
* 支持自动刷课，鼠标离开不会暂停。
* 播放完毕，自动跳转下一节
* 支持顺序刷课，和,只刷未完成的课2种模式
* 支持代刷，点击进入脚本界面的连接即可，各种网课3块钱一门
